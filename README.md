
# C++ Password Generator 

A simple and fast secure password generator written in C++. An educational project utilizing the modern C++11 `<random>` library.

## Features
* Generates a password of any user-defined length.
* Uses the Mersenne Twister algorithm (`mt19937`) to ensure high-quality randomness.
* Includes uppercase and lowercase letters, numbers, and special characters.

## How to run
To compile and run the program, you need a C++ compiler (e.g., GCC, Clang).

Type the following in your terminal:
```bash
g++ main.cpp -o password_generator
./password_generator
